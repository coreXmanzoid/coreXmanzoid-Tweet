About ChatFlick

Built from passion,
one commit at a time.

ChatFlick is a full-stack social microblogging platform crafted as a year-long academic and personal project — combining modern web development, Python backend engineering, and a genuine love for building things that work.

Our Mission

A space to share, connect, and express.

ChatFlick was built with one goal in mind: to create a clean, functional platform where people can share short thoughts, follow others, and engage with content — without the noise.

This project is an honest demonstration of what a developer can build from scratch — authentication, real-time notifications, an AI assistant, and a polished UI — all woven together through months of focused work.

What's Inside

Core Features

ChatFlick covers the full social platform experience — from account management to real-time notifications — implemented end-to-end by a single developer.

🔐

Authentication

Email & password login, Google OAuth, CAPTCHA protection, email verification, and secure password reset via tokenized links.

📝

Posts & Feed

Create, edit, and delete posts with hashtag extraction and @mention parsing. Browse a personalized "Following" feed or discover random content.

❤️

Engagement

Like, repost, share, and comment on posts. Milestone notifications fire at 100, 1K, and 10K likes or reposts to celebrate your reach.

🔔

Notifications

In-app and Firebase push notifications for new followers, likes, comments, reposts, mentions, and milestone events — with deduplication built in.

🤖

Manzoid-AI Assistant

An integrated Groq-powered AI chat assistant to help you draft posts, generate hashtags, summarize content, or just answer questions — contextually aware per session.

⚙️

Settings & Privacy

Full profile and account management: password change, privacy toggles, notification preferences, theme switching, and an admin dashboard for moderation.

Under the Hood

Technology Stack

Every layer of ChatFlick was chosen deliberately — a cohesive stack that balances developer productivity with production capability.

Flask (Python)
Core backend framework — routing, blueprints, session management, and template rendering.
SQLAlchemy + PostgreSQL
ORM-backed relational database for users, posts, follows, comments, and notifications.
Flask-Login & Authlib
Session-based authentication and Google OAuth 2.0 integration.
Firebase Cloud Messaging
Push notification delivery for real-time follower and engagement alerts.
Cloudinary
Profile picture storage and transformation with automatic resizing and format optimization.
Groq (Llama 3.1)
Fast LLM inference powering the Manzoid-AI assistant with per-session conversation history.
Jinja2 + Bootstrap 5
Server-side templating with a responsive, component-driven front-end baseline.
jQuery + Vanilla JS
Dynamic UI interactions, AJAX-driven feeds, optimistic updates, and real-time comment loading.
